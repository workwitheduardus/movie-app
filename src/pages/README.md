# Pages

Folder ini untuk menyimpan halaman-halaman utama aplikasi.

## Struktur yang Disarankan

Berdasarkan design Figma, buatlah halaman-halaman berikut:

- `HomePage.tsx` - Halaman utama dengan list movies
- `MovieDetailPage.tsx` - Halaman detail movie
- Dan halaman lainnya sesuai kebutuhan

## Tips

1. Gunakan React Router untuk routing
2. Setiap page component bisa memiliki sub-components di folder components
3. Gunakan custom hooks dari folder hooks untuk data fetching
4. Implementasikan animations dengan Framer Motion
